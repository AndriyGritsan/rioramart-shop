# 🛒 RioraMart Shop

RioraMart is a simple eCommerce web application built with Django.  
It allows users to browse products, add items to cart, and place orders.

---

## 🚀 Features

- Product listing with categories
- Product detail pages
- Shopping cart functionality
- Checkout form with billing & shipping details
- Order creation and storage
- Admin panel for managing products, orders, and banners
- Banner system (main & side banners)
- Blog section
- Wishlist functionality

---

## 🛠️ Tech Stack

- Python 3
- Django
- SQLite / PostgreSQL (depending on setup)
- HTML, CSS, JavaScript
- Bootstrap (for UI styling)
